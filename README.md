# Zomato Data Analysis (EDA)

**Overview:**
This project performs Exploratory Data Analysis (EDA) on a Zomato dataset to understand patterns in restaurant ratings, pricing, locations, and cuisines.

**Objectives:**
* Analyze rating distribution of restaurants
* Study relationship between cost and ratings
* Understand factors affecting restaurant success

**Dataset:**
The dataset contains restaurant details such as name, location, cuisines, cost for two, and ratings.

**Key Insights:**
* Most restaurants have ratings between 3.5 and 4.0
* Higher price does not necessarily mean better ratings
* Certain locations have higher-rated restaurants
* Popular cuisines dominate the dataset

**Visualizations:**
* Rating distribution
* Cost vs rating analysis
* Cuisine popularity
* Location-based trends

**Tech Stack:**
* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook (VS Code)

**Learning Source:**
This project was inspired by a tutorial from Krish Naik.
I implemented the analysis and explored the dataset to understand the concepts better.

**How to Run:**
1. Clone the repository
2. Open the notebook in VS Code or Jupyter Notebook
3. Install required libraries
4. Run all cells

**Conclusion:**
The analysis shows that restaurant success is influenced more by location and cuisine rather than just pricing. Ratings are generally clustered in a narrow range, indicating competitive standards across restaurants.
